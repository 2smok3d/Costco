🔑 Supervisor — Register Procedures ▼
Override
  • Supervisor key in Override Mode
  • <CLEAR> → <OVERRIDE> → enter supervisor # → <ENTER>
  • Key in password → <ENTER>
Override Terminal Secured
  • <SIGN ON/OFF> → <CLEAR> → Override
Verify Price & Linked Item Record
  • <6> → <SIGN ON/OFF>
  • Scan or key item(s) → <TOTAL>
  • To exit: <SIGN ON/OFF> twice
Pressing ENTER twice prints item detail.
Void Linked Fee
  • Sign onto register → <6> → <SIGN ON/OFF> → scan or key item → <ENTER> twice
  • Use four-digit link # on receipt OR look up item # in controller
  • <VOID> → key linked fee followed by eight zeroes
Tender Cashing — Check
  • <1> → <SIGN ON/OFF> → scan membership #
  • Key in $ amount → <CHECK>
  • Endorse check, change due member
Tender Cashing — Executive 2% Reward
  • <1> → <SIGN ON/OFF> → scan membership #
  • <6> → </> → <MISC TENDER>
  • Scan or key barcode → <ENTER>
  • Endorse reward, change due member
Tender Cashing — Citi Reward (US)
  • <1> → <SIGN ON/OFF> → scan membership #
  • Scan reward OR <9> → </> → <MISC TENDER>
  • Key entire bar code → <ENTER> → change due member
Tender Exchange / Correction
  • <2> → <SIGN ON/OFF> → scan membership #
  • <VOID> → key in $ amount
  • Press incorrect (previous) tender: <CASH>, <CHECK>, or <STANDALONE>
  • Key in $ amount → select new tender type
  • Deface receipt, attach to Daily Register Sheet
Coupon Adjustment
  • Scan membership # → <SLIP PRINT> (if receipt available) → <ADJUSTMENT>
  • <VOID> → scan coupon → <TOTAL>
  • Tender, override required
  • Deface adjustment receipt, attach to Daily Register Sheet
Coupons outside valid date require an override.
Training Mode
  • Sign on with Training Operator Code (500–505), password: 9999
  • <7> → <SIGN ON/OFF> → exit <SIGN ON/OFF> twice
  • All receipts must be defaced and attached to Daily Register Sheet
Training Mode does NOT affect IAFEP numbers.
Price Override
  • Key in $ amount → <PRICE>
  • Scan or key in item number → <ENTER> → override
Department Ring
  • Only use when item number is not set up
  • Key in $ amount → <PRICE>
  • Key in Department # → <DEPARTMENT>
  • Key in item number (use zero if none) → <ENTER> → override required
Void Previous Transaction — Same Register, Same Day
  • Hold <CTRL> → <ADJUSTMENT>
  • Enter transaction # → <ENTER>
  • Enter transaction amount → <ENTER>
  • Tender key twice → override
  • Deface receipt, attach to Daily Register Sheet with original receipt
Void Previous Transaction — Adjustment Register, Same Day
  • Hold <CTRL> → <ADJUSTMENT>
  • Scan transaction barcode OR enter register number → <ENTER>
  • Enter transaction # if register number keyed → <ENTER>
  • Enter transaction amount → <ENTER>
  • Tender key twice → override
  • Deface receipt, attach to DRS with original
Adjustments
  • Scan or key original membership #
  • Press <ADJUSTMENT>
  • Scan/key items to be voided → <TOTAL>
  • Tender key → override
  • Deface adjustment receipt, note original purchase date(s) if different day
  • Attach to Daily Register Sheet
Transfer Register to Another
  • Sign on as Sup/Mgr at NEW register
  • <8> → <SIGN ON/OFF>
  • Key in previous register # → <ENTER> → re-enter register # → <ENTER>
  • <ENTER> for same operator OR <TOTAL> for current operator
  • Receipt will reprint → continue with order → <TOTAL> → tender → sign off
  • Check problem at failed register and correct it

🖥 Controller ▼
Department Totals Report (Real-Time Sales)
  • TGCS ACE APPLICATION [1] → Reports [R] → Sales Report → Department Totals Report
  • Select report parameters and destination
Outstanding Store & Forward Report
  • IBM ACE Applications → Reports [R] → SurePOS ACE EPS Reports → Outstanding SAF/TOR Report
  • Run to ensure transactions approved when EFT was down have been released
View Offline Pending Report
  • IBM ACE Applications → Accounting → Scroll to Close → Arrow down to System Status → [ENTER]
  • Active transactions may delay auto-store close
Loan Report
  • IBM ACE Applications [1] → Reports [R] → Listing and Log Reports [L] → Exception Log → ENTER
  • Non-cash recycler: verify against CFIs from vault
  • Cash recycler: should show NO loans
Reprint Receipts — Current Day (ACE Transaction Search)
  • TGCS ACE APPLICATION [1] → Manager Procedures [M] → ACE Transaction Search Tool [A]
  • Manager Journal [M] → search by appropriate filters
Supervisor or manager must accompany member using reprinted receipt to exit.
View Summary Journals (Previous Day Receipts)
  • TGCS ACE [1] → Manager Procedures [M] → View Summary Journals [V] → Sort by Date
  • Press Enter on first terminal → in Search field: member# followed by *
  • Use arrow down to navigate → Press Print
Only 15 lines before and 14 after highlighted line will print. Long receipts require multiple prints.
Operator Maintenance — Adding an Operator
  • TGCS ACE [1] → Data Maintenance [D] → Operator Authorization [O]
  • Key cashier number → ENTER → select model: *1=Cashier, *2=Training, *3=Refund, *4=Inventory, *5=Sales Audit, *6=Supervisor, *7=Manager
  • Enter cashier name as FIRST*LAST → enter initial password → select Expire Password
Enhanced Password (Supervisors & Managers)
  • Click CHANGE ENHANCED PASSWORD or press ALT+C
  • Enter 8-character alpha-numeric password → verify → SAVE → OK → F3 to exit
Enhanced Password Rules
  • Expires every 90 days
  • Must be alpha-numeric (at least one letter and one number)
  • Case sensitive · Must be 8 characters
  • No more than 2 letters or numbers in sequence or repeated
  • No spaces or special characters (&, @, %)
  • Cannot reuse previous 5 passwords
Change Food Court Lookup Keys
  • TGCS ACE [1] → User Applications [U] → Keyboard Lookup Utility [K] → Whse Food Court → ENTER
  • TAB between keys to edit/delete/enter Food Court item numbers
  • ALT+S to save changes
Changes upload to register after signing off.

💳 Cashier — Payment & Transactions ▼
Change Expired Password
  • Register displays "Password Expired Enter New Password"
  • Key new password → ENTER → verify new password → ENTER
  • "B070 New Password Accepted" → CLEAR
Passwords expire every 90 days. Never share passwords.
Change Cashier Password (Not Expired)
  • Key Cashier # → </> → current password → OVERRIDE
  • Enter new password → SIGN ON → verify → ENTER
  • "B070 New Password Accepted" → ENTER
EBT Food (US)
  • Member swipes EBT card, selects EBT Food
  • "E" on receipt = EBT eligible items
  • For EBT subtotal: TOTAL → EBT → TOTAL
  • TOTAL → DEBIT/CREDIT to tender
Never enter member's PIN on their behalf.
EBT Cash (US)
  • Member swipes EBT card → selects EBT Cash
  • TOTAL → CREDIT/DEBIT to tender → member enters PIN
FSA
  • "F" on receipt = FSA eligible items
  • For FSA subtotal: <2> → </> → TOTAL
  • Less than FSA total: key FSA amount → CREDIT/DEBIT
Purchase Orders
  • SLIP PRINT on → scan items → TOTAL → PO TENDER → PO TENDER
  • ENTER to charge 1.5% admin fee OR CLEAR to waive fee
  • Override required → insert document to slip print
Loan
  • LOAN → key in $ amount → CASH → TOTAL
Pickup — Cash
  • PICKUP → key dollar amount → CASH → TOTAL
Pickup — Check
  • PICKUP → CHECK → TOTAL
Executive 2% Rewards included with checks. Do NOT void a check pickup.
Resale
  • Fill out resale card → RESALE (on) → scan items → RESALE (off)
  • Resale can toggle on/off multiple times
  • TOTAL → tender → print onto resale card → member must sign
Void Transaction — Dollar Amount
  • VOID → TOTAL → override required
  • Deface receipt and attach to Daily Register Sheet
Void Transaction — Zero Dollar Amount
  • VOID → TOTAL
  • Deface receipt and attach to Daily Register Sheet
Voiding Partial EFT Tender
  • TOTAL → VOID → CREDIT/DEBIT (voids previous EFT payment)
Void EFT payments BEFORE performing a void total to ensure funds return correctly.
Costco Wallet
  • Scan digital membership QR code → ring items → TOTAL
  • Linked Credit Card (US): CTRL+IC → IC
  • Shop Card: CTRL+STANDALONE
Point of Sale Activated Cards (POSA)
  • Scan barcode → cashier retains card until order tendered successfully
  • Error message: press CLEAR → cashier keeps non-activated cards
  • Activation receipt prints for each POSA card
Only way to void POSA cards is Void Previous transaction.
Operator Statistics
  • NO SALE → 96 → SIGN ON
  • Prints: Operator Name, IPM (items per minute), MPH (members per hour)
AS400 option IAFEP displays operator statistics for registers 1–50 the following day.
Converting Register to Refund Mode (Dec 1 – Jan 10)
  • Sign on as Refund Clerk, Sup, or Mgr on selected FE register
  • SEASONAL REFUND → ENTER → register is now in Refund mode
  • Sign off → have refund cashier sign on
  • To exit during day: SEASONAL REFUND → ENTER → sign off

🛒 Self Checkout (SCO) ▼
Open the Lane
  • Scan Shopper Assistant (SA) Card
  • Select OPEN LANE – CANCEL PENDING TRANSACTIONS
Age Validation
  • Scan SA card → age validation screen shows → verify ID
  • Scan back of ID OR key member date of birth
  • If member appears over 40: key generic DOB 01011961
  • ENTER
Suspend Transaction
  • Member presses ASSISTANCE NEEDED → scan SA card
  • VOID OR SUSPEND → TRANSACTION → SUSPEND TRANSACTION
  • Suspended receipt prints with retrieval barcode
  • SA or Supervisor retains receipt and takes member to register
Void Transaction
  • Member presses ASSISTANCE NEEDED → scan SA card
  • VOID OR SUSPEND → TRANSACTION → VOID TRANSACTION
  • Override required using established procedures
Void Single Item
  • Member presses ASSISTANCE NEEDED → scan SA card
  • VOID OR SUSPEND → SINGLE ITEM → select item → VOID ITEM
Remove voided items from scale.
Troubleshooting — SCO Lane Frozen
  • Scan SA card on tabletop or hand scanner
  • If SA card doesn't scan: press green button behind or underneath SCO touch screen
  • Select CLOSE LANE → scan SA card → OPEN LANE → continue
Zero Security Scale
  • Scan SA card → NEXT → ZERO SECURITY SCALE
🔧 Troubleshooting — Registers & Devices ▼
Register Quick Reload
  • S1 (hold CTRL + press DEPT + release) → 91
  • S2 (hold CTRL + press RESALE + release)
  • [T0010 ENTER REQUEST] displays → 99
  • S2 again — register will reload
U008 Message
  • Power register OFF — hold power button 10 seconds
  • Wait additional 10 seconds → turn ON — register will reload
Z001/Z002 Message
  • Enter register address as 4 digits (e.g., #1 = 1001, #16 = 1016, #501 = 1501)
  • S2 (hold CTRL + press RESALE + release) — register will reload
Readdress Full Screen (Touchscreen)
  • Full screen shows Z001/Z002 with keypad and S1/S2 buttons
  • Enter register # as 4 digits → S2
  • Message: "Z044 format desk, to format disk key S1, else key S2 press" → S1
  • Once format complete → S2 → Z012 appears → reboot takes 10–15 minutes
Register System Down
  • Answer alarms on controllers, respond to message, contact Help Desk
  • Entire system DOWN: call HOTLINE (425) 313-8001 option 9
Store & Forward (EFT System Down)
  • Credit cards up to $1,000.00 can process via Store & Forward
  • Once communication restored, stored transactions forward automatically
  • NO DEBIT or DMC transactions at registers — must use standalone
  • ALL credit over $1,000 and ALL DEBIT: TOTAL → STANDALONE → STANDALONE → receipt prints → hand to supervisor
PIN Pad Reset — MX915
  • Press 1+5+9 simultaneously → release → X → 1
  • PIN pad reloads (~10 seconds)
PIN Pad Reset — M400
  • Press 1+5+9 simultaneously → release → 1+5+9 again to reach "Supervisor" screen
  • Press red X → select Reboot → press green O
Credit/Debit Cards Not Swiping
  • Magnetic reader may need cleaning — obtain cleaning card, follow instructions
  • Clean PIN pad keys with damp towel only
🏧 Standalone (US) ▼
Sale Transaction
  • Press SALE → enter Clerk # → REGISTER # → ENTER
  • If "Clerk ID Doesn't Exist": enter Yes to add register, password 1111
  • Enter $ Amount → ENTER
Refund Transaction
  • Press REFUND → password 1111 → ENTER → enter $ Amount → ENTER
  • Enter Clerk # → REGISTER # → ENTER
Shop Card Balance Inquiry
  • From main screen: NEXT twice → BALANCE INQUIRY → swipe Shop Card
  • Balance prints on receipt
Adding a Register to Standalone
  • All V400M standalones preprogrammed for registers 1–10
  • From home screen: CANCEL (red X) → NEXT × 3 → SETUP
  • Password 1111 → ENTER → CLERKS/SERVERS → ADD → enter new register # → ENTER → CANCEL
Standalones automatically settle at 10pm local time. If warehouse tries to run settlement, Standalone generates a declined message — payment still received.

  • Tender damaged/lost card as payment via manual entry
Balance transfers by phone must be done from a warehouse landline.
