Meat-Specific Notes
• Variable-weight (catch-weight) items: UPCs starting with 2 encode weight/price per GS1 retail convention. • Production yield: sub-primal in → retail cut out → trim/fat/bone = shrink. Target 70–85% yield on beef depending on grade/trim spec. • Shrink reason codes: Out-of-date · Spoilage · Trim/Fat/Bone · Damage · Contamination · Markdown · Internal Use · Donation (Feeding America) · Recall/Hold. • Receiving accuracy: variable-weight cases individually weighed or scanned. Total scanned weight should equal invoice ±0.5%. • COOL compliance: Country-of-Origin required on imported chicken, lamb, and goat muscle cuts. Country code in item master prints automatically on scale label. • 13 physical inventories per year (one per 4-week fiscal period). Pre-inventory prep: print expected on-hand from IAI opt 1 for every active SKU. • Costco company-wide shrink target: 0.11–0.12% of sales. Fresh meat runs higher — track weekly with STK/DSL to stay within department tolerance.

The 30-Second Item Walk

Chain inquiries instead of looking at one screen alone:

IAI → opt 1 (on-hand by warehouse) → opt 3 (sales by region) → exit → RHI (receiving history) → DSL (dept context)

Five screens, one minute. You now know more about that SKU than 90% of your peers.

Daily Routine — Fresh Dept Lead

Before doors open: Sign on → DSL (yesterday sales vs. plan vs. LY) → LSDT (sales + labor variance) → RHI (any item received yesterday, confirm posting) → IAI/IIL on top-10 SKUs → walk floor with EDSR printout → build morning-huddle talking points.

Mid-shift: Re-check on-hand for fast movers → submit transfer-in via IIL if OOS imminent → resolve locked-record errors → print exception spools.

Closing: Final DSL/LSDT read → WRKSPLF to clear and route day's spool files → key shrink/damages/donations → sign off cleanly.

Weekly Routine

Mon: 7-day DSL by SKU, rank top-25 and bottom-25. Identify slow movers with cover days >5.

Tue: Weekly RHI variance review for the truck cycle.

Wed: Comp-shop maintenance — particularly tobacco, liquor, seasonal price audits.

Thu: EDSR review for upcoming weekend pricing.

Fri: Pre-weekend OOS sweep. Place transfer-in requests with sister warehouses via IIL.

Sun/EoW: Print weekly DSL spool, file in dept binder, prep weekly variance commentary.

Period (4-Week) Routine

3 days before close: Reconcile all RHI receivings for the period. Post all damages, donations, and transfers. Reconcile in-transit pallets.

Count day: Print expected on-hand from IAI opt 1 for every dept SKU → conduct physical count → key variances. Biggest variances always live in slow-mover bins.

Day after close: Review shrink % vs. target (0.11–0.12% company-wide). Document root cause for every variance over threshold. Print period DSL + turn calc + shrink summary as one-pager for AGM/GM.

Day 1 new period: Re-baseline cover days. Reset standing orders if YoY trend has shifted ±10% or more.

Talk Like a Buyer — Justify-an-Order Template

"Item _____, in dept ____, showed cover days of N (vs. target ~3 days for fresh). DSL shows -X% YoY. RHI shows we're still receiving at last year's rate. Recommend cutting the standing receive by Y cases. Estimated annual savings in shrink: $_____."

Frame in dollars, not units: 30 extra cases of ribeye = $X tied up plus spoilage risk. That's the language that moves decisions.

Career Advancement — SIT Path

Path: Hourly → SIT (Supervisor In Training, 12-week) → Supervisor → Merchandise Training Course (12-week) → Dept Manager → AGM → GM. ~85% of managers started hourly.

Merit increases: Automatic every 1,040 hours worked in $1/hr increments up to top-of-scale (~5 years to top out). Discretionary raises at SIT entry and above.

SIT interview talking points using AS400:

• "I used LSDT/LSWT to analyze planned vs. projected sales and adjusted staffing accordingly."

• "I caught a receiving discrepancy worth $__ using RHI vs. physical count."

• "I reduced cover days from X to Y on top movers without OOS using IAI and IIL."

• "I trained [N] team members on AS400 item lookups and sign printing."

Common Error Messages & Fixes
CPF5032
Record already locked to this job
F12 to back out. If persistent: sign off and back on. IT can release via WRKOBJLCK if needed.
CPF4101
File not found / level check on open
Check library list with DSPLIBL. Usually IT pushes a fix — sign back on after they confirm.
CPF5027
Record being used by another job
Wait and retry. If blocking operations, IT can ENDJOB the offending session.
CPF2189
Job not authorized to use library
Your profile lacks access. Manager contacts IT to grant library access.
CPF9999
Function check — unhandled program error
Press F3 or Enter. Run DSPJOBLOG and capture the CPF code. Send to IT — they need the exact error from the job log.
U008
Register communication error
Power register OFF, hold power 10 seconds. Wait 10 more seconds. Power ON — register reloads.
Z001 / Z002
Register address error
Enter register # as 4 digits (register 1 = 1001, register 16 = 1016, register 501 = 1501). Press S2 (hold CTRL + RESALE + release).
