# Dept 61 — Nightly Close Report

Mobile-optimized close report template for Dept 61, Costco Tempe.

Fill out each field, tap **Generate Text**, then **Copy & Send** to paste directly into a text message.

## Fields
- Conv. Poultry
- Poultry Grinds
- OG Poultry
- Grinds
- Bricks
- Cryo
- Date Watch
- Case Condition
- Notes
- Out at

## Usage
Open `index.html` in any browser. Works offline once loaded.
Live via GitHub Pages: `https://YOUR-USERNAME.github.io/dept61-close`
